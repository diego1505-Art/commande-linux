    Tab — autocomplétion des commandes et fichiers

    Ctrl + L — effacer l'écran
    Ctrl + Z — mettre une commande en pause

    Règles importantes

    Linux est sensible à la casse — Documents ≠ documents
    Pas de corbeille — rm supprime définitivement
    Ne jamais copier‑coller dans le terminal sans vérifier
    sudo — nécessaire pour les dossiers système
    root (#) — super‑admin, à utiliser avec précaution
    ~/ — raccourci pour /home/kali

    Astuces Bash avancées

    commande1 && commande2 — exécute commande2 seulement si commande1 réussit
    commande1 || commande2 — exécute commande2 seulement si commande1 échoue
    commande & — lancer en arrière‑plan
    commande > fichier — rediriger la sortie (écrase)
    commande >> fichier — rediriger la sortie (ajoute)
    commande 2> erreurs.txt — rediriger les erreurs
    !! — répéter la dernière commande
    sudo !! — répéter la dernière commande en admin
